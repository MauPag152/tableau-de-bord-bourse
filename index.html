📝 Méthodologie et Interprétation

1. Rendements Logarithmiques

Les rendements journaliers sont calculés en rendements logarithmiques (log-returns) pour leur additivité temporelle :

r(t) = ln( P(t) / P(t−1) )

2. Volatilité Annualisée (Règle √252)

La volatilité annuelle est obtenue en annualisant l'écart type journalier via la racine du nombre de jours de trading :

σ_annuelle = σ_journalière × √252

Convention : une année boursière ≈ 252 jours de cotation ; hypothèse de rendements i.i.d.

3. Régression Linéaire (Moindres Carrés Ordinaires)

Ŷ(t) = a × t + b   (t = 1 … N)

a = pente (variation moyenne du prix par jour de trading), b = ordonnée à l'origine.

4. Erreur Type de la Régression

SE = √[ Σ(Y(t) − Ŷ(t))² / (N − 2) ]

5. Interprétation du Canal

  • Régression (Noire) : tendance centrale du modèle linéaire.
  • ±1 SE (Orange / Bleue) : contient ≈ 68 % des observations — évolution « normale ».
  • ±2 SE (Rouge / Verte) : contient ≈ 95 % des observations — un dépassement signale une situation statistiquement extrême (surachat/survente) pouvant indiquer un retour à la moyenne ou une rupture de tendance.
